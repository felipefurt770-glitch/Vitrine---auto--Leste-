Você é um desenvolvedor front-end sênior trabalhando no site da Vitrine Auto Leste.

Contexto do projeto:
A Vitrine Auto Leste é uma vitrine de venda direta de veículos selecionados, com foco em captação de leads pelo WhatsApp, anúncios orgânicos, SEO local e apresentação premium dos carros. Não é para parecer marketplace genérico de vários anunciantes nesta primeira versão; deve soar como uma operação comercial com atendimento direto.

Arquivos que estou te entregando:
- index.html: protótipo funcional em HTML/CSS/JS puro.
- data/vehicles.json: mapa dos veículos com dados completos.
- data/vehicles.csv: planilha/CSV dos veículos para importação e conferência.
- assets/cars/: imagens organizadas por pasta/slug de veículo.

Número oficial do WhatsApp para todos os botões e formulários:

Formato para links: 5521979112975.

Requisitos obrigatórios:
1. Fazer o site funcionar perfeitamente, sem erro de JavaScript, sem dependência de build, sem npm obrigatório na primeira versão.
2. Manter a identidade visual da Vitrine Auto Leste: azul-marinho escuro, dourado, branco premium, visual automotivo premium, limpo, comercial e confiável.
3. Os filtros devem funcionar perfeitamente:
   - Busca textual por marca, modelo, versão, motor, cor, câmbio, combustível e descrição.
   - Marca.
   - Categoria.
   - Câmbio.
   - Combustível.
   - Cor.
   - Preço mínimo.
   - Preço máximo.
   - KM máximo.
   - Ano mínimo.

4. Se um veículo estiver com KM não informado, ele deve aparecer normalmente quando o filtro de KM estiver vazio. Quando o usuário filtrar KM máximo, veículos sem KM devem sair do resultado para não gerar dado falso.
5. Todos os botões de WhatsApp devem abrir conversa com mensagem pré-preenchida sobre o veículo correto.

7. Cada veículo deve ter galeria com imagens da pasta correspondente. Não misturar fotos de veículos diferentes.
8. O site deve ser responsivo para desktop e mobile.
9. Otimizar SEO básico:
   - title e meta description.
   - headings bem organizados.
   - alt nas imagens.
   - textos com palavras-chave naturais: veículos selecionados, carros usados, Zona Leste, financiamento, WhatsApp, hatch, SUV, sedan, utilitário etc.

   - Gol: R$ 48.900

12. O site deve abrir localmente ao clicar no index.html. Se você modularizar, garanta que o ambiente local esteja bem explicado.
13. Se converter para React/Vite/Next no futuro, preservar todos os dados, filtros, lógica de WhatsApp e estrutura de imagens.
